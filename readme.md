# Jay's PWA Builder
=======

## Installation
1. clone the git repo.
2. navigate to the project directory and run.
    npm install

## CLI Usage
    node app.js --init
    node app.js --list
    node app.js --run <project-name> [--port <specific port number>]
    node app.js --serve

## User-Interface Usage
    npm run start
